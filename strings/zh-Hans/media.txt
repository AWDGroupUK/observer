Media Menu

  Media: 媒体
  Upload: 上传媒体

Admin Menu

  Media Settings: 媒体设置

Upload Media

  Heading: 上传媒体
  Instructions: 点击灰色框下方，或将文件拖动到上传框。最大文件大小为1GB。
  Select Files Legend: 选择文件
  Select Files: 单击选择文件，或拖动和拖放在这里的文件。
  File Drop Failed: 无法获取文件信息。试着点击选择文件来代替。
  Upload Status: 上传：%1%
  ID3 Unavailable: ID3 / EXIF​​数据不可用
  Wait For Upload To Save: 无法保存，请等到所有上传完成。

Edit Media

  Heading: 编辑媒体
  Instructions: 编辑根据需要进行以下，然后单击保存按钮提交更改。

Edit Media Form

  ID3 Button: 使用ID3数据
  Upload Queue: 文件在上传队列

  Artist: 艺术家
  Title: 称号
  Album: 专辑/系列
  Year: 年
  Category: 类别
  Country: 国家
  Language: 语言
  Genre: 类型
  Comments: 评论
  Copyright Owner: 你是版权拥有者
  Media Approved: 被媒体认可
  Status: 状态
  Dynamic Selections: 包括动态的选择

  Private Option: 私人（提供给您只）
  Public Option: 公众（提供给大家）
  Not Applicable Option: 不适用

  Copy Field Button: 复制到所有

  Media Saved: 媒体已保存。

  Media Not Found: 此媒体项目已不存在。
  Upload Not Valid: 该文件上传无效。
  File Upload Required: 一个文件上传需要新媒体。
  Format Not Supported: 不支持这种文件格式。
  Required Fields Not Filled: 一个或多个必填字段未填写。
  Artist/Title Too Long: 一个或多个艺术家/标题字段是太长。
  Year Not Valid: 这一年是无效的。
  Category Not Valid: 选定的类别不再有效。
  Country Not Valid: 选定的国家不再有效。
  Genre Not Valid: 所选体裁不再有效。
  Language Not Valid: 选择的语言将不再有效。
  Status Not Valid: 媒体的地位是无效的。
  Genre Not Valid For Category: 所选体裁不适用于所选类别。

Media Settings
  Heading: 媒体类别和流派
  Instructions: 点击一个类别名称来显示该类别的类型。
  Add Category Button: 添加类别
  Add Genre Button: 添加一个流派

  Supported File Formats: 支持的文件格式
  File Format Instructions: 选择您想要接受媒体上传的文件格式。更改此设置将不会影响现有的媒体。

  Audio Formats: 音频格式

  FLAC Description: （自由无损音频编解码器）
  MP3 Description: （MPEG-1第3层音频）
  MP4 Description: （MPEG-4音频）
  OGG Description: （奥格容器格式）
  WAV Description: （波形音频）

  Video Formats: 视频格式
  AVI Description: （音频视频交错，容器格式）
  MPEG Description: （运动图像专家组）
  WMV Description: （Windows媒体视频）
  MOV Description: （苹果的Quicktime）

  Image Formats: 图像格式
  JPEG Description: （联合图像专家组）
  PNG Description: （可移植网络图形）
  SVG Description: （可缩放矢量图形）

  Formats Saved Message: 文件格式设置保存。

  No Genres: 没有找到体裁。
