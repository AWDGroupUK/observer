Media Menu

  Media: สื่อ
  Upload: อัพโหลดสื่อ

Admin Menu

  Media Settings: การตั้งค่าสื่อ

Upload Media

  Heading: อัพโหลดสื่อ
  Instructions: คลิกที่กล่องสีเทาด้านล่างหรือลากไฟล์ลงบนกล่องที่จะอัปโหลด ขนาดไฟล์สูงสุด 1GB
  Select Files Legend: เลือกไฟล์
  Select Files: คลิกเพื่อเลือกไฟล์หรือลากและวางไฟล์ที่นี่
  File Drop Failed: ไม่สามารถได้รับข้อมูลไฟล์ ลองคลิกเลือกไฟล์แทน
  Upload Status: อัพโหลด: %1%
  ID3 Unavailable: ID3 / EXIF​​ ข้อมูลไม่พร้อมใช้งาน
  Wait For Upload To Save: ไม่สามารถบันทึกโปรดรอสักครู่จนกว่าภาพที่ส่งทั้งหมดจะเสร็จสมบูรณ์

Edit Media

  Heading: แก้ไขสื่อ
  Instructions: แก้ไขต่อไปตามที่ต้องการจากนั้นคลิกที่ปุ่มบันทึกที่จะกระทำการเปลี่ยนแปลงของคุณ

Edit Media Form

  ID3 Button: ใช้ข้อมูล ID3
  Upload Queue: ไฟล์ที่อยู่ในคิวอัปโหลด

  Artist: ศิลปิน
  Title: ชื่อเรื่อง
  Album: อัลบั้ม / ซีรีส์
  Year: ปี
  Category: หมวดหมู่
  Country: ประเทศ
  Language: ภาษา
  Genre: ประเภท
  Comments: ความคิดเห็น
  Copyright Owner: คุณเป็นเจ้าของลิขสิทธิ์
  Media Approved: สื่อได้รับการอนุมัติ
  Status: สถานะ
  Dynamic Selections: รวมถึงในการเลือกแบบไดนามิก

  Private Option: ภาคเอกชน (ที่มีให้คุณเท่านั้น)
  Public Option: สาธารณะ (ใช้ได้กับทุกคน)
  Not Applicable Option: ไม่สามารถปรับใช้

  Copy Field Button: คัดลอกไปทั้งหมด

  Media Saved: สื่อได้รับการบันทึก

  Media Not Found: รายการสื่อนี้ไม่มีอยู่แล้ว
  Upload Not Valid: อัปโหลดไฟล์ไม่ถูกต้อง
  File Upload Required: อัปโหลดไฟล์เป็นสิ่งจำเป็นสำหรับสื่อใหม่
  Format Not Supported: รูปแบบไฟล์นี้จะไม่ได้รับการสนับสนุน
  Required Fields Not Filled: หนึ่งหรือมากกว่าหนึ่งสาขาที่จำเป็นต้องได้รับไม่เต็ม
  Artist/Title Too Long: หนึ่งหรือมากกว่าหนึ่งศิลปิน / สาขาที่ชื่อยาวเกินไป
  Year Not Valid: ปีที่ไม่ถูกต้อง
  Category Not Valid: ประเภทที่เลือกไม่ถูกต้อง
  Country Not Valid: เลือกประเทศที่ไม่ถูกต้อง
  Genre Not Valid: เลือกประเภทไม่ถูกต้อง
  Language Not Valid: ภาษาที่เลือกไม่ถูกต้อง
  Status Not Valid: สถานะสื่อไม่ถูกต้อง
  Genre Not Valid For Category: เลือกประเภทที่ไม่สามารถใช้ได้สำหรับประเภทที่เลือก

Media Settings
  Heading: สื่อและหมวดหมู่ประเภท
  Instructions: คลิกที่ชื่อหมวดหมู่เพื่อแสดงประเภทที่อยู่ในประเภทที่
  Add Category Button: เพิ่มหมวดหมู่
  Add Genre Button: เพิ่มประเภท

  Supported File Formats: การสนับสนุนรูปแบบไฟล์
  File Format Instructions: เลือกรูปแบบไฟล์ที่คุณต้องการที่จะยอมรับสำหรับการอัปโหลดสื่อ เปลี่ยนการตั้งค่านี้จะไม่ส่งผลกระทบต่อสื่อที่มีอยู่

  Audio Formats: รูปแบบเสียง

  FLAC Description: (Free Lossless แปลงสัญญาณเสียง)
  MP3 Description: (MPEG-1 Layer-3 เสียง)
  MP4 Description: (MPEG-4 เสียง)
  OGG Description: (รูปแบบ Ogg คอนเทนเนอร์)
  WAV Description: (สัญญาณเสียง)

  Video Formats: รูปแบบวิดีโอ
  AVI Description: (Audio Video สอดใบแทรก, รูปแบบภาชนะ)
  MPEG Description: (Moving Picture Experts Group ซึ่ง)
  WMV Description: (Windows Media Video)
  MOV Description: (Apple Quicktime)

  Image Formats: ภาพรูปแบบ
  JPEG Description: (Joint ถ่ายภาพ Experts Group)
  PNG Description: (แบบพกพาเครือข่ายกราฟิก)
  SVG Description: (กราฟิกเวกเตอร์ Scalable)

  Formats Saved Message: ตั้งค่ารูปแบบไฟล์ที่บันทึกไว้

  No Genres: ประเภทไม่พบ


Category Edit

  Name: ชื่อ
  Name Required Message: ชื่อหมวดหมู่จะต้อง
  Must Remove Message: คุณต้องเอาประเภทที่อยู่ในหมวดหมู่นี้ก่อนที่จะลบหมวดหมู่
  Saved Message: หมวดหมู่ที่บันทึกไว้
  Category Deleted Message: หมวดหมู่ลบ

  Edit Category: แก้ไขหมวดหมู่
  New Category: หมวดหมู่ใหม่

Genre Edit

  New Genre: ใหม่ประเภท
  Edit Genre: แก้ไขประเภท

  Name: ชื่อ
  Description: ลักษณะ
  Category: หมวดหมู่

  Delete Genre Confirm: คุณแน่ใจหรือว่าต้องการลบประเภทหรือไม่

  Name Required Message: ชื่อประเภทที่จำเป็น
  Genres Updated Message: ประเภทการปรับปรุง
  Genre Deleted Message: ประเภทที่ถูกลบ
