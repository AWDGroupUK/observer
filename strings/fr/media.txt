Upload Media

  Heading: Transférer multimédia
  Instructions: Cliquez sur la case grise ci-dessous, ou faites glisser les fichiers sur la boîte à télécharger. Taille maximale du fichier est de 1 Go.
  Select Files Legend: Sélectionnez Fichiers
  Select Files: Cliquez sur pour sélectionner les fichiers, ou faites glisser et déposer des fichiers ici.
  File Drop Failed: Impossible d'obtenir les informations de fichier. Essayez de cliquer pour sélectionner les fichiers à la place.
  Upload Status: L'ajout: %1%
  ID3 Unavailable: ID3 / Données EXIF Non disponible
  Wait For Upload To Save: Impossible d'enregistrer, s'il vous plaît attendre jusqu'à ce que tous les téléchargements sont terminés.
