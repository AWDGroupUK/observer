  Items Heading: Articles playlist
  Items Instructions Advanced: médias de glisser vers la zone ci-dessous. Double-cliquer sur les éléments à définir les propriétés. Cliquez sur pour sélectionner un élément, puis supprimez / retour arrière pour supprimer, touches haut / bas pour se déplacer.
  Items Instructions Standard: Faites glisser les médias et playlists de la région ci-dessous. Double-cliquer sur les éléments à définir les propriétés. Cliquez sur pour sélectionner, puis supprimer / retour arrière pour supprimer, touches haut / bas pour se déplacer. Vous pouvez également cliquer-glisser pour réorganiser.

  Advanced Playlist Dynamic Selection Not Supported: Sélections dynamiques et des identifications de la station ne sont pas pris en charge par des listes de lecture avancées. Ces éléments ont été ignorées.

  Station ID: ID station

  Total Duration: Durée totale

  LiveAssist Buttons Heading: LiveAssist Boutons
  LiveAssist Buttons Instructions: Faites glisser listes de lecture pour ce domaine de programmer la zone du bouton de LiveAssist. Chaque playlist sera une colonne de boutons, avec un seul bouton par élément de la playlist.

  Clear Items Confirm: Désactivez tous les éléments de la liste?
  Change Playlist Type Confirm: Modification du type de playlist effacera la liste de lecture existante. Êtes-vous sûr de vouloir faire cela?

  Clear LiveAssist Items Confirm: Désactivez tous les boutons de LiveAssist?

  Unable To Edit: Impossible de modifier cette playlist.
  Name Required: Un nom de playlist est nécessaire.
  Valid Status Required: Un état de validité est nécessaire.
  Valid Type Required: Un type de validité est nécessaire.

  Item Not Valid: Un ou plusieurs éléments de playlist est pas valide.
  Durations Not Valid: Un ou plusieurs durées de médias est invalide ou nul.
  Approved Media Required: Seulement approuvé, les médias non archivées peuvent être utilisés dans des listes de lecture.
  Cannot Use Private Item: Un élément de média est marqué comme privé. Il ne peut être utilisé dans les listes de lecture créées par le même propriétaire.
  Dynamic Item Not Valid: Un ou plusieurs éléments dynamiques de playlist est pas valide.

  LiveAssist Item Not Valid: Un ou plusieurs listes de lecture LiveAssist bouton est invalide.

  Error While Saving: Une erreur est survenue lors de l'enregistrement de cette playlist.
