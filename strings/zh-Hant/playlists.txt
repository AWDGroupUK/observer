Playlists Menu

  Playlists: 播放列表
  New: 新建播放列表

Playlist New

  Heading: 新建播放列表
  Instructions: 填寫以下信息，播放列表，然後將媒體和播放列表中的項目到下面的列表區。

Playlist Edit

  Heading: 編輯播放列表
  Instructions: 根據需要編輯以下播放列表。

  Select One Playlist Only: 只選擇一個播放列表。

  Form Heading: 播放列表數據
  Name: 名字
  Description: 描述
  Status: 狀態
  Type: 類型

  Private Option: 私人（提供給您只）
  Public Option: 公眾（提供給大家）

  Standard Option: 標準
  Advanced Option: 先進
  LiveAssist Option: LiveAssist

  Items Heading: 播放列表項
  Items Instructions Advanced: 拖動媒體到下方的區域。雙擊項目設置屬性。點擊選擇項目，然後刪除/退格刪除，上/下鍵來移動。
  Items Instructions Standard: 拖動媒體和播放列表下方的區域。雙擊項目設置屬性。單擊以選中，然後刪除/退格刪除，上/下鍵來移動。您也可以點擊拖動來重新排序。

  Zoom In: 放大
  Zoom Out: 縮小

  Drag Items Here: 拖動項目在這裡

  Add Dynamic Selection Button: 添加動態選擇
  Add Station ID Button: 添加站ID
  Add Breakpoint Button: 添加斷點

  Dynamic Selection: 動態選擇
  Dynamic Selection Last Search Notice: 動態選擇是根據你過去的媒體搜索。

  Advanced Playlist Dynamic Selection Not Supported: 動態選擇和站號是不是先進的播放列表支持。這些項目已被忽略。

  Breakpoint: 斷點

  Station ID: 站ID

  Total Duration: 總工期
  Total Duration Estimated: （估計由於可變長度的內容）

  LiveAssist Buttons Heading: LiveAssist按鈕
  LiveAssist Buttons Instructions: 播放列表拖動到該區域進行編程LiveAssist按鈕區域。每個播放列表將是按鈕的一列，與每個播放列表項目的一個按鈕。

  Clear Items Confirm: 清除播放列表中的所有項目？
  Change Playlist Type Confirm: 更改播放列表類型將清除現有的播放列表。你確定要這麼做嗎？

  Clear LiveAssist Items Confirm: 清除所有LiveAssist按鈕？

  Unable To Edit: 無法編輯該播放列表。
  Name Required: 播放列表名稱是必需的。
  Valid Status Required: 一個有效的狀態是必需的。
  Valid Type Required: 一個有效的類型是必需的。

  Item Not Valid: 一個或多個播放列表項是無效的。
  Durations Not Valid: 一個或多個媒體持續時間是無效的或零。
  Approved Media Required: 只有獲得批准，未歸檔介質可以在播放列表中使用。
  Cannot Use Private Item: 媒體項目被標記為私有。它只能在由相同的所有者創建的播放列表中。
  Dynamic Item Not Valid: 一個或多個動態播放列表中的項目是無效的。

  LiveAssist Item Not Valid: 一個或多個LiveAssist按鈕的播放列表是無效的。

  Error While Saving: 同時節省了此播放列表時發生錯誤。

  Heading: 動態項目屬性
  Selection Name: 選擇名稱
  Number Of Items: 項目數
  All Items: 所有項目
  Image Duration: 圖片時間
  In Seconds: （秒）

  Description Heading: 描述

  Advanced Search Type: 高級搜索
  All Media Search Type: 所有媒體
  Standard Search Type: 搜索（藝術家或標題）

  Number Of Items Invalid: 項目的數量是無效的。
  Image Duration Invalid: 的圖像的持續時間是無效的。
  Invalid Search Criteria: 無效的搜索條件。
